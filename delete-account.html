Users of the StudyStack app can request deletion of their account and associated data at any time.

How to request account deletion

  1. Send an email to: your-email@example.com
  2. Use the subject line: StudyStack Account Deletion Request
  3. Include the email address used to register in the app

What data is deleted

  • User account information (email / user ID)
  • Associated app data stored in Firebase

Data retention

No personal data is retained after deletion, except where required by law.

Processing time

Account deletion requests are processed within 7 days.
